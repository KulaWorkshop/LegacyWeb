Tools

Here you will find a list of tools that are useful for modding the games.


Our Tools

  • Kula Workshop Level Viewer - A web-based level viewer for viewing Kula World level files.
  • Kula World PAK Tool - A program for extracting and creating Kula World .PAK files.

Not Our Tools

  • QuickBMS - For extracting and injecting files.
  • HxD - For hex editing files.
  • Hex Workshop - For hex editing files. (With data visualizer)
  • Ghidra - For reverse engineering and analyzing the games code.
  • Ghidra PSX LDR - For adding PSX support to Ghidra.
  • PSX MODE2 - For replacing files within the games .bin files.
  • PVV - For viewing PSX VRAM to rip textures.
  • EmuHawk - For debugging and reverse engineering the games.
  • no$psx - For debugging and reverse engineering the games.
  • jpsxdec - For extracting music and video files.
  • PSound - For extracting sound files.
